# 🎬 Movie Recommendation System

A content-based movie recommendation web application built using **Machine Learning** and **Streamlit**. The system suggests movies similar to a selected movie using text-based similarity techniques.

---

## 🚀 Live Demo
👉 https://movie-recommender-1258.streamlit.app/

---

## 📌 Project Overview

This project recommends movies based on their similarity using content-based filtering. It analyzes movie metadata such as titles, genres, and descriptions to compute similarity scores and suggest relevant movies.

---

## 🧠 How It Works

1. Movie dataset is preprocessed (cleaning + feature engineering)
2. Text features are converted into vectors using **TF-IDF**
3. Similarity between movies is calculated using **Cosine Similarity**
4. The most similar movies are recommended based on user selection
